What's is Git and Github ?


Git dan Github ini adalah sebuah tools wajib bagi developer, karena dengan bantuan kedua tools ini developer dapat berkolaborasi satu sama lain. Dimana git adalah perangkat lunak pengontrol versi atau proyek manajemen kode perangkat lunak, sedangkan github adalah layanan penginapan web bersama untuk proyek pengembangan perangkat lunak yang menggunakan sistem pengontrol versi Git dan layanan hosting.

Oke, sekarang kita akan mencoba untuk membuat project sederhana menggunakan git dan github. Sebelum membuat projectnya, kita harus menginstall git nya terlebih dahulu. karena untuk proses upload project kita ke github harus menggunakan git. Mari kita install git nya terlebih dahulu. Let's go.

Review kode git : 
            
1. git init => untuk proses inisialisasi folder local
2. git add <file> => untuk menambahkan file
3. git status => untuk melihat status file
4. git commit -m "pesan/comment" => untuk memberikan pesan atau komentar pada file
5. git push origin master => untuk proses upload file ke github.
