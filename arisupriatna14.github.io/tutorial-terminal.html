What's is Command Line ?


Command line? pasti sudah tidak asing lagi ditelinga kita. Nah, sekarang apa itu Terminal? jangan berpikir terminal bus ya? hehe. Jadi Command lain dan Terminal itu sama saja. Hanya untuk penyebutan terminal sering disebutkan di os unix. Command line atau terminal yaitu text interface untuk sistem operasi komputer. Command line banyak digunakan dikalangan programmer atau developer, karena dengan menggunakan command line semua pekerjaan terselesaikan dengan cepat, khususnya untuk dibidang server. Jika anda masih awam bagaimana cara menggunakan command line, mari kita belajar dari basic terlebih dahulu. Silahkan perhatikan kode-kode perintah command line berikut.
  1. $ pwd

    2. pwd digunakan untuk menampilkan nama direktori kerja saat ini. Perhatikan gambar dibawah ini.

    style="font-weight: bold; font-size: 20px;" Pada gambar, ketika kita mengetikan kode perintah pwd, maka muncul nama direktori kerja kita saat ini, yaitu berada di /home/ari.

  2. $ ls

    3. ls digunakan untuk menampilkan nama file dan nama folder kita. Perhatikan gambar berikut ini.

    Pada gambar,ketika kita mengetikkan kode perintah ls,maka muncul nama folder dan nama file. Teks yang berwarna biru itu adalah nama folder, sedangkan teks yang warna kuning adalah nama file.

  3. $ cd <nama_folder>

    4. cd digunakan untuk memindahkan direktori yang anda tentukan. Misalnya seperti berikut :

    Pada gambar di atas, saya melakukan kode perintah cd Documents/ artinya saya masuk ke direktori dengan nama Documents. Untuk kembali ke folder sebelumnya cukup ketikkan kode perintah ($ cd .. )

  4. $ mkdir <nama_folder>

    5. mkdir berfungsi untuk membuat folder baru. Jadi dengan menggunakannya kode ini di terminal, membuat folder baru lebih simple. Contohnya perhatikan gambar berikut :

    Pada gambar diatas, kita membuat folder baru dengan nama latihan-1 didalam folder latihan-cli.

  5. $ touch <nama_file>

    6. touch <nama_file> digunakan untuk membuat file baru didalam direktori kerja. Contoh :

    Pada gambar diatas, kita membuat file dengan nama index.html di folder latihan-cli. Sekarang didalam folder latihan-cli terdapat satu folder dengan nama latiha-1 dan file dengan nama index.html.
